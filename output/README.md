# Outagamie County Growth Potential – Google Earth Overlay

**File:** `Outagamie_County_Growth_Overlay.kml`

## How to Use
1. Download the KML file from GitHub
2. Open in **Google Earth** (desktop or web)
3. Toggle layers on/off to see the growth pattern

## Color Legend
| Color | Growth Rating | Municipalities |
|-------|--------------|----------------|
| 🟢 Green | **High** | Greenville, Grand Chute, Harrison, Combined Locks, Little Chute |
| 🟡 Yellow-Green | **Moderate to High** | Buchanan, Freedom, Kimberly |
| 🟡 Yellow | **Moderate** | Vandenbroek, Hortonville, Wrightstown |
| 🟠 Orange | **Low to Moderate** | Center, Ellington, Bovina, Cicero, Dale, etc. |
| 🔴 Red | **Low** | Black Creek, Oneida Twp, Bear Creek, Nichols, Shiocton |

## Key Insight
The green cluster around Appleton and Kaukauna shows exactly where the Cody Bjugan / North Star framework targets — villages and towns directly outside city limits with utility infrastructure and future land use maps showing development pressure.
